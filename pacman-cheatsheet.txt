yay
--mflags                          - передача флага из yay в makepkg
--mflags --skipinteg              - не запускать интеграционные тесты source (sha суммы) при сборке приложения из aur

-Qi имя_пакета                    - информация о установленном пакете
-Si имя_пакета                    - информация о пакете

-Rscn $(yay -Qdtq)                - удаление пакетов сирот
-Scc                              - удаление всех кешей пакетов включая yay

-Q                                - список установленных пакетов
-Qm                               - список пакетов установленных вручную

-Ql                               - список файлов с составе пакета
-Qo                               - поиск пакета которому принадлежит файл

pactree -s -d1 -o1 имя_пакета     - строит граф зависимости пакета
pactree -r -s -d1 -o1 имя_пакета  - строит граф зависимости этого пакета для других пакетов
